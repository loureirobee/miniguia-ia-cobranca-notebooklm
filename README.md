# Miniguia de Estudos com NotebookLM
## IA Aplicada à Cobrança e Recuperação de Crédito

### Sobre o Projeto
Este projeto foi desenvolvido como parte do desafio da DIO para explorar o uso do NotebookLM como ferramenta de aprendizagem ativa.
O objetivo foi estudar como Inteligência Artificial, Automação, Chatbots, Voicebots e Modelos Preditivos podem ser aplicados em operações de cobrança e recuperação de crédito, utilizando fontes abertas e estruturando o conhecimento em um caderno temático.

---

# Objetivos de Estudo
* Entender os principais conceitos de IA aplicados à cobrança.
* Identificar oportunidades de automação em jornadas de recuperação de crédito.
* Compreender o uso de IA Generativa em negociações digitais.
* Estudar modelos preditivos para propensão de pagamento.
* Construir um material reutilizável para futuras consultas.

# Miniguia de Estudos

## Resumo Estruturado
### O que é Inteligência Artificial
Conjunto de técnicas capazes de simular comportamentos humanos para análise, tomada de decisão e automação.

### IA Generativa
Modelos capazes de criar conteúdo novo a partir de dados previamente aprendidos.
Exemplos:
* ChatGPT
* Gemini
* Claude

### IA Aplicada à Cobrança
Aplicações:
* Chatbots de negociação.
* Voicebots.
* Segmentação de clientes.
* Score de recuperação.
* Recomendação de ofertas.
* Análise de sentimento.

### Benefícios
* Redução de custos.
* Atendimento 24x7.
* Escalabilidade.
* Aumento da recuperação.
* Melhor experiência do cliente.

### Riscos
* Dados incorretos.
* Viés algorítmico.
* Falta de supervisão humana.
* Questões relacionadas à LGPD.

# Glossário

## IA
Inteligência Artificial.

## IA Generativa
Tecnologia capaz de criar novos conteúdos.

## LLM
Large Language Model.

## NLP
Processamento de Linguagem Natural.

## Chatbot
Assistente virtual baseado em texto.

## Voicebot
Assistente virtual baseado em voz.

## Machine Learning
Método de aprendizado a partir de dados.

## Score de Cobrança
Indicador que estima a probabilidade de recuperação.

## LGPD
Lei Geral de Proteção de Dados.

## RAG
Técnica que combina busca de conhecimento com IA generativa.
